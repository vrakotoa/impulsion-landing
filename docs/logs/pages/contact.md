# Page Contact

## Décisions

- La page `/contact` contient un formulaire Formspree natif au site.
- Le formulaire collecte prénom, nom, email, téléphone optionnel, message, et une case RGPD obligatoire.
- Formspree est utilisé comme service externe de réception des messages.
- Le formulaire garde un lien secondaire vers Tidycal pour réserver un appel.

## Intégrations

- Formspree endpoint : `https://formspree.io/f/xwvdddzd`
- Tidycal : `https://tidycal.com/verorakotoarisoa/30-minute-call`
